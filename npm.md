# NPM

## Criação projeto

Chama o prompt

```bash
npm init
```

Cria direto o pacote npm

```bash
npm init -y
```

## Install -g

Instala globalmente na máquina o pacote npm.

```bash
npm i -g @ionic/cli

npm install --global @ionic/cli
```

## Install -D

Instala localmente a dependência no projeto.

```bash
npm i -D @ionic/cli

npm install --save-dev @ionic/cli
```

## Parar processamento do node

```bash
ctrl + c
```
